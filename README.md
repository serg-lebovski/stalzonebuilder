# StalZone Builder

Веб-приложение для оптимизации сборок артефактов в игре STAL ZONE.

## Возможности

- Актуальные данные из репозитория [EXBO-Studio/stalzone-database](https://github.com/EXBO-Studio/stalzone-database)
- Ручная сборка: выбор костюма + контейнера + артефактов, расчёт итоговых статов
- **Авто-подбор** оптимальной сборки из твоего инвентаря под цель (Скорость / Пулестойкость / Приведёнка)
- Формула Приведёнки: `(Макс.ХП + Пулестойкость) × (1 + Живучесть% / 100)`
- Инвентарь хранится в PostgreSQL (аккаунты пользователей)

## Запуск на сервере

```bash
# Первичная установка
bash scripts/setup.sh

# Или через Python-скрипт с Windows
python scripts/deploy.py
```

## Запуск локально (разработка)

```bash
python -m app.main --server --port 8080
```
Открыть: http://localhost:8080

## Конфигурация

Переменные окружения:

| Переменная    | По умолчанию | Описание                          |
|---------------|--------------|-----------------------------------|
| `PORT`        | `8080`       | Порт HTTP-сервера                 |
| `SERVER_MODE` | `1`          | Не открывать браузер при старте   |
| `DB_URL`      | —            | PostgreSQL DSN                    |
